# AI Agents Domain Axioms

## AX-DOM-AI-AGENTS-001: Human in the Loop

AI agents should not autonomously execute destructive or irreversible actions without human confirmation.

## AX-DOM-AI-AGENTS-002: Explainability

Agent decisions should be explainable. When an agent makes a non-obvious choice, it should articulate the reasoning.

## AX-DOM-AI-AGENTS-003: Guardrails Are Mandatory

Every agent must operate within defined guardrails. No agent may disable or bypass its own guardrails.
