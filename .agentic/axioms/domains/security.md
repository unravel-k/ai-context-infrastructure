# Security Domain Axioms

## AX-DOM-SECURITY-001: Principle of Least Privilege

Every component should operate with the minimum privileges necessary.

## AX-DOM-SECURITY-002: Defense in Depth

Never rely on a single security control. Layer multiple controls.

## AX-DOM-SECURITY-003: Assume Breach

Design systems with the assumption that a breach will occur. Plan for containment and recovery.
