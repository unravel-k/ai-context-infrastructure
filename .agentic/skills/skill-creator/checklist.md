# Skill Creator Checklist

- [ ] Asked user what skill to create or modify
- [ ] Inspected `.agentic/skills/` for existing matches
- [ ] Inspected `.claude/skills/` for existing adapters
- [ ] Decided create vs modify with user confirmation
- [ ] Created/updated `skill.md` with complete workflow
- [ ] Created/updated `axioms.md` with skill-specific axioms
- [ ] Created/updated `checklist.md` with verification steps
- [ ] Created/updated `output.md` with output format
- [ ] Created/updated `.claude/skills/<skill>/SKILL.md` adapter
- [ ] Asked user about route registration
- [ ] Updated `.agentic/router/routes.yaml` if confirmed
- [ ] Returned summary in creator-result format
- [ ] Offered to run validation
