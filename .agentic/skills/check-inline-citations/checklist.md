# Check Inline Citations Checklist

- [ ] Received the draft to check
- [ ] Identified all factual claims in the text
- [ ] Checked each claim for an inline citation
- [ ] Classified each claim: properly cited, bibliography-only, missing, unclear
- [ ] Checked citation placement (point of use, not deferred)
- [ ] Did not add or fabricate any citations
- [ ] Returned coverage report with percentages and recommendations
