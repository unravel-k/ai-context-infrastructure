# Verify Citations Checklist

- [ ] Received the draft text to verify
- [ ] Extracted all citations with URLs, titles, authors, and supported claims
- [ ] Verified each URL resolves (or confirmed broken/hallucinated)
- [ ] Checked content match for each resolved URL
- [ ] Checked title/author accuracy for each citation
- [ ] Classified every citation: verified, broken link, hallucinated, misattributed, or unverifiable
- [ ] Flagged or removed problematic citations
- [ ] Preserved claims where possible (only removed if unsupported)
- [ ] Returned verification report with complete classification breakdown
- [ ] Stated overall reliability qualitatively
