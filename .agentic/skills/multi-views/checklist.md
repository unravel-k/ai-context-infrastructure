# Multi-Views Checklist

- [ ] Received and confirmed full topic list with user
- [ ] Ran for-and-against independently per topic (not cross-contaminated)
- [ ] Collected all per-topic outputs with sources preserved
- [ ] Identified cross-cutting themes (shared catalysts, divergences, correlations)
- [ ] Built side-by-side comparison table
- [ ] Ranked topics qualitatively (strongest for, weakest for, strongest against, most balanced)
- [ ] Every ranking includes a direct reason about that topic
- [ ] Produced consolidated output with per-topic summaries + comparison + ranking
- [ ] Did not fabricate scores or conclusions
- [ ] Preserved all source attributions from per-topic runs
