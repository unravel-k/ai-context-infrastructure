# Axiom Creator Checklist

- [ ] Asked user for the axiom or principle
- [ ] Classified as universal / domain-specific / job-specific / skill-specific / project-specific
- [ ] Selected correct target file based on classification
- [ ] Inspected target file for similar axioms
- [ ] Handled duplicates (modify / append / skip) with user confirmation
- [ ] Assigned unique axiom ID following convention
- [ ] Wrote axiom to target file with correct format
- [ ] Asked user about route integration
- [ ] Updated `.agentic/router/routes.yaml` if confirmed
- [ ] Returned summary with classification, file, ID, route changes, unresolved questions
