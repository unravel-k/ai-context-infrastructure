# Frontend Slides Checklist

- [ ] Asked user about presentation content and tone
- [ ] Planned slide structure (from scratch) or extracted PPTX content (from file)
- [ ] Generated 3 visual style previews for user to choose from
- [ ] User selected a style by viewing, not describing
- [ ] Generated single self-contained HTML file
- [ ] Included viewport-base.css for responsive layout
- [ ] Applied animation patterns where appropriate
- [ ] Avoided generic AI aesthetics
- [ ] Offered deployment (Vercel) or PDF export
- [ ] Verified the HTML opens correctly in a browser
