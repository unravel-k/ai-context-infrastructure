# Frontend Slides Axioms

See `.agentic/axioms/skills/frontend-slides.md` for the canonical axiom definitions.

## AX-SKILL-FRONTEND-SLIDES-001: Visual Choice Over Verbal Description

Never ask the user to describe design preferences in words. Generate visual previews and let them choose by seeing. Non-designers cannot articulate aesthetic preferences.

## AX-SKILL-FRONTEND-SLIDES-002: Single Self-Contained File

Output must be a single self-contained HTML file with no external dependencies, no npm build step, and no framework. The file must work in any modern browser and remain functional for years.

## AX-SKILL-FRONTEND-SLIDES-003: Avoid Generic AI Aesthetics

Avoid default AI-generated aesthetics: no purple gradients on white backgrounds, no generic stock-photo styling. Use the curated presets from STYLE_PRESETS.md.
